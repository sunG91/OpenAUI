# 安全公告

## 概述

本文档用于发布 Open AUI 项目的安全相关说明、已知风险与漏洞披露流程。

## 安全说明

### 数据与权限

- **API Key**：存储于本地 `backend/data/`，请勿将包含敏感数据的目录提交至版本控制
- **连接密钥**：用于 WebSocket 认证，建议使用强密钥并定期更换
- **本地执行**：Open AUI 可在用户电脑上执行命令、操作 GUI、访问文件，请仅在可信环境中使用

### 建议

1. 不要在生产环境使用默认密钥（如 `openaui-default-key`）
2. 定期更新依赖，关注安全公告
3. 打包后的 exe 运行于本地，不主动外连（除配置的 API 调用外）
