# 产品设计

## 文档

完整产品设计文档见：[open-aui-产品设计文档](https://github.com/sunG91/OpenAUI/blob/main/docs/open-aui-产品设计文档.md)

## 核心模块

- **语音模块**：实时监听、唤醒、TTS
- **控制台模块**：命令执行、安全控制
- **浏览器模块**：DOM 解析、脚本操作、多模态
- **任务拆分**：意图理解、模块调度、模型调度
- **记忆模块**：行为记录、上下文管理
- **扩展模块**：JS/Py 脚本、安全沙箱

## 模式

- **Chat**：纯对话模式
- **AUI**：AI 操作模式，可执行控制台、浏览器、扩展等

## 技术栈

- Electron + React + Tailwind CSS
- Node.js 后端
- nut-tree / RobotJS（GUI）
- Playwright（浏览器）
